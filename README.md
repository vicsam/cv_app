# cv_app

A Flutter app, that displays personal information in the form of a CV.

## Getting Started

- Setup Flutter SDK
- Setup Dart SDK
- Install Visual Studio Code (vs code)
- Install flutter and dart extensions in vs code
- Clone the repo
- Run the program
